# Workshop 05 - Building a REST API with Express and MongoDB
## Context
In this workshop, the goal was to make a REST API routes for various CRUD operations for a "blog" database:
- POST (/): create a blog post
- GET (/): get all posts
- GET (/:id): get one post by id
- PUT (/:id): update a post by id
- DELETE (/:id): delete a post by id

For testing the API routes, I used Postman (VS Code extension).

Among the other things I learned in this workshop building the API routes, I also learned basic Mongoose operations and also schema validation.
